## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 16.x or later)
- npm or yarn package manager

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/newtonfrank/newton-portfolio-v1.git
   cd newton-portfolio-v1
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open your browser to [http://localhost:5173](http://localhost:5173) to view the application

### Environment Variables

This project doesn't require any environment variables, but you can add a `.env` file if needed:
- Create a `.env` file in the root directory
- Add any environment-specific variables if required
